Gradebook
=========

## What is Gradebook?

Gradebook is a simple grade calculator, using HTML5, jQuery, and [Shake.js](https://github.com/alexgibson/shake.js/). Data is modeled on a 40-40-20 grading scale. 

Built as part of the App Development Course (M257).
